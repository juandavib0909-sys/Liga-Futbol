## Funcionalidades
- Leer configuración desde `config.txt`
- Registrar partidos
- Mostrar tabla de posiciones
- Guardar tabla en `tabla.txt`
- Mostrar historial de jornadas
- Mostrar partidos jugados
- Uso de archivos txt
- Uso de structs, vectores, funciones y punteros

## Estructura
- `main.cpp`: código principal
- `data/config.txt`: configuración de liga y equipos
- `data/partidos.txt`: partidos registrados
- `data/fechas.txt`: historial de jornadas
- `data/tabla.txt`: tabla exportada

Compilar:

```bash
g++ src/main.cpp -o liga.exe
